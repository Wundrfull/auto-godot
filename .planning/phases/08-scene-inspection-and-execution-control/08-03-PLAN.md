  <action>
    Create `src/gdauto/debugger/execution.py` with async execution control functions.

    All functions follow the fire-and-forget pattern using the public `session.send_fire_and_forget()` method (added in Plan 08-01 Task 2). This sends protocol commands without creating pending futures, because execution control commands do not receive matching responses; Godot confirms state changes via unsolicited debug_enter/debug_exit messages handled by the session's recv loop.

    After sending, each function updates local state proactively (game_paused, current_speed) to avoid race conditions (the recv loop may not process the confirmation before we return).

    1. **async def pause_game(session: DebugSession) -> GameState:**
       - `await session.send_fire_and_forget("scene:suspend_changed", [True])`
       - Set `session.game_paused = True`
       - Return `GameState(paused=True, speed=session.current_speed, frame=0)`

    2. **async def resume_game(session: DebugSession) -> GameState:**
       - `await session.send_fire_and_forget("scene:suspend_changed", [False])`
       - Set `session.game_paused = False`
       - Return `GameState(paused=False, speed=session.current_speed, frame=0)`

    3. **async def step_frame(session: DebugSession) -> GameState:**
       - Per D-10 discretion: auto-pause if game is running
       - If `not session.game_paused`: send suspend_changed [True] first via send_fire_and_forget, set game_paused = True
       - Then send next_frame: `await session.send_fire_and_forget("scene:next_frame", [])`
       - Return `GameState(paused=True, speed=session.current_speed, frame=0)`
       - Note: game stays paused after stepping

    4. **async def set_speed(session: DebugSession, multiplier: float) -> GameState:**
       - Validate: if multiplier <= 0.0, raise DebuggerError(message=f"Speed multiplier must be positive, got {multiplier}", code="DEBUG_INVALID_SPEED", fix="Use a positive number (e.g., debug speed 10)")
       - `await session.send_fire_and_forget("scene:speed_changed", [multiplier])`
       - Set `session.current_speed = multiplier`
       - Return `GameState(paused=session.game_paused, speed=multiplier, frame=0)`

    5. **def get_speed(session: DebugSession) -> GameState:**
       - Pure local query, no protocol message
       - Return `GameState(paused=session.game_paused, speed=session.current_speed, frame=0)`
  </action>
